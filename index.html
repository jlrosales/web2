¿Qué es la programación?

Muchas veces se piensa que programar es solo escribir código. Este es un concepto erróneo, ya que programar implica analizar y solucionar problema, idear y planificar como llegar a esa solución y finalmente, crear el algoritmo que se codificará para materializar la solución a ese problema.

Algoritmo

La definición es: “Conjunto ordenado y finito de operaciones que permite hallar la solución de un problema" (Real Academia Española, RAE). En otras palabras, es detallar los pasos de manera lógica y ordenada para solucionar un problema. Es importante destacar que un algoritmo siempre será finito, ósea, tendrá un paso de inicio y, siguiendo la secuencia lógica y ordenada de pasos, siempre tendrá un final, que será la solución al problema planteado. Un ejemplo de algoritmo son las indicaciones e instrucciones en secuencia que aparecen en un manual.

Representación de un algoritmo

Un algoritmo puede ser representado de las siguientes formas:

  • Lenguaje Natural

    Instrucciones dadas en el lenguaje que utilizamos (español para este caso) con una secuencia lógica que soluciona un problema, por ejemplo, una receta de cocina.

  • Diagrama de flujo

    Nos permite detallar los pasos de nuestro algoritmo, siendo un punto intermedio entre el lenguaje natural y el lenguaje de programación .

Pensamiento lógico

El pensamiento lógico es indispensable a la hora de programar, ya que nos permite reconocer y analizar los problemas y posibles soluciones. Nos permitirá anticiparnos a los diferentes comportamientos e interacciones que se puedan presentar y programar un código que tenga en cuenta todas estas situaciones. Mientras más desarrollado se encuentre el pensamiento lógico, más a menudo se logrará que el programa haga lo que esperamos que haga y más rápido se obtendrán soluciones a los problemas diarios que se presenten.

Java

Es un lenguaje de programación y una plataforma informática comercializada por primera vez en 1995 por Sun Mycrosystems (actualmente pertenece a Oracle). Java se puede usar en casi todo tipo de plataforma, desde portátiles hasta centros de datos, consolas para juegos, teléfonos móviles e internet. Gracias a su JVM es capaz de ejecutarse en múltiples plataformas. Es un lenguaje estable y maduro, muy demandado, con amplias librerías y soporte.

¿Por qué utilizar Java?

Java es la base para prácticamente todos los tipos de aplicaciones de red, además del estándar global para desarrollar y distribuir aplicaciones móviles, juegos, contenido basado en web y software de empresas. Cuenta con 9 millones de desarrolladores en el mundo. 3 mil millones de teléfonos móviles ejecutan Java, entre otras cosas. Además, según la encuesta anual de desarrolladores de Stack Overflow, Java se encuentra dentro de los 5 lenguajes de programación más populares y, según el índice TIOBE 2020, Java se encuentra en el 3er lugar de los lenguajes de programación más utilizados. También vale la pena destacar que importantes empresas ocupan Java, tales como Twitter, Netflix, Uber, Spotify, entre otras

Ventajas de elegir Java

  • Escribir software en una plataforma y ejecutarla en otra (con su JVM)
  • Escribir aplicaciones para teléfonos móviles, productos de consumo, procesadores remotos y prácticamente cualquier dispositivo electrónico.
  • Gran comunidad.
  • Gran cantidad de librerías.
  • Principal lenguaje de Android.
  • Lenguaje con amplia trayectoria.

Si te interesa profundizar más puedes ir al sitio oficial de Java.com.

Instalación y Configuración del entorno de trabajo

Para comenzar a desarrollar software con Java lo primero que necesitamos es tener instalado todas las tecnologías que nos acompañarán en este proceso.

  • JDK: (Java Development Kit) o Kit de desarrollo en Java. Contiene todo lo necesario para poder trabajar con el lenguaje, entre eso el JRE (Java Runtime Environment) o entorno de ejecución y la JVM (Java Virtual Machine) o máquina virtual de Java que es el entorno en el cual se ejecutan los programas.
  • IDE: (Integrated Development Enviroment) o Entorno de Desarrollo Integrado que será nuestro editor de texto (Las ventajas de usar un IDE son que cuenta con un depurador y una interfaz gráfica que nos ayudará en el desarrollo de nuestras aplicaciones. también nos mostrará nuestros errores de sintaxis, entre otras). Para este curso utilizaremos Eclipse.

Puedes seguir los pasos de instalación de java y del entorno de desarrollo integrado eclipse en la guía de instalación.

Teniendo configurado nuestro entorno de trabajo podemos crear nuestro primer programa.
